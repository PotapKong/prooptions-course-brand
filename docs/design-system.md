# Design System

## Суть

Курс должен располагать новичка к изучению опционов.

У человека не должно возникать ощущения, что его затягивают в “трейдинг-сигналы”. Должно возникать ощущение: здесь спокойно разложат сложный инструмент, покажут риск и научат читать позицию.

## Главный стиль

**Premium Trading Desk**

- тёмный терминал;
- аккуратная аналитика;
- мягкая премиальность;
- реальная рыночная фактура;
- без офисной вычурности.

## Композиционный ритм

На каждом визуале должен быть один главный объект:

- лицо Сэма;
- payoff-график;
- карточка Call / Put;
- option chain;
- strike line;
- схема риска.

Не смешивать всё сразу.

## Сетка

Использовать 12-колоночную сетку для 16:9 и 4-колоночную сетку для вертикальных обложек.

Минимальные поля:

- 16:9: 7% ширины;
- 1080x1350: 9% ширины;
- 9:16: 10% ширины сверху и снизу, чтобы не попадать в слепые зоны.

## Светлая учебная система

Для PDF, домашних заданий и конспектов использовать светлые блоки.

Формула:

- бумажный фон `#F7F4EC`;
- графитовый текст;
- зелёные маркеры;
- золотые важные блоки;
- красный только для риска.

## Примеры заголовков

- `ОПЦИОНЫ С НУЛЯ`
- `CALL И PUT БЕЗ МАГИИ`
- `КАК РАБОТАЕТ СТРАЙК`
- `РИСК ВИДНО НА ГРАФИКЕ`
- `ЧТО ДЕЛАЕТ THETA`
- `ЗАЧЕМ НУЖНА ВОЛАТИЛЬНОСТЬ`

## Нежелательные формулировки

- `секретная стратегия`
- `заработай без риска`
- `иксы на опционах`
- `пассивный доход`
- `сигналы от трейдера`
