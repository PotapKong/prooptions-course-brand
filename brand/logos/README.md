# Logos

Сюда добавить официальные логотипы.

Правила:

- не перерисовывать логотипы;
- не генерировать логотипы заново через image model;
- не растягивать и не искажать;
- использовать только проверенные исходники;
- держать SVG/PNG отдельно от референсов.

Рекомендуемая структура:

```text
brand/logos/pro-options/
brand/logos/sam/
brand/logos/partners/
```
